# DevOps ToDo App 🚀

Простой ToDo-менеджер, развернутый с использованием Docker и Nginx. 
Создан в обучающих целях для демонстрации базовых DevOps-навыков.

## 📋 О проекте

Это веб-приложение на Flask с SQLite базой данных, которое:
- Позволяет создавать/удалять задачи
- Отмечать задачи выполненными
- Показывает информацию о контейнере и сервере

## 🛠 Технологии

- **Python/Flask** — backend приложения
- **SQLite** — база данных
- **Docker** — контейнеризация приложения
- **Docker Compose** — оркестрация контейнеров
- **Nginx** — reverse proxy
- **Bash** — скрипты автоматизации

## 🏗 Архитектура
Пользователь → Nginx (порт 80) → Flask App (порт 5000) → SQLite (/data)
